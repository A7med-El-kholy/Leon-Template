# Leon Template Project

Hey! This is a simple **HTML & CSS project** I built called **Leon Template**.  
It's a clean and responsive template for practicing layouts, styling, and smooth scroll effects.

## Features

- Fully **responsive** design  
- **Flexbox & Grid** layouts  
- Smooth scroll for navigation  
- Simple hover effects on buttons and links
